AutoFlow

Automated File Organization System using Rust

Project Overview

AutoFlow is a command-line based file automation tool developed using the Rust programming language. The main purpose of this project is to automatically organize files in a directory based on their file types such as documents, images, videos, and other categories.

This project helps users save time and reduce manual effort when managing large numbers of files on their system.

Problem Statement

Many users store files in a single folder such as the Desktop or Downloads directory. Over time, this leads to clutter, difficulty in finding files, and poor file management. Manual sorting of files is time-consuming and inefficient.

AutoFlow solves this problem by automatically detecting file types and moving them into appropriate folders without user intervention.

Objectives of the Project

  • To automate file organization using a command-line interface
  • To reduce manual effort in managing files
  • To demonstrate system automation using Rust
  • To provide a fast, reliable, and secure solution
  • To apply real-world use of file handling concepts

Key Features

  • Automatic file detection based on extensions
  • Creates folders dynamically if they do not exist
  • Moves files safely without data loss
  • Works through a simple command-line interface
  • Fast execution due to Rust’s performance

Technologies Used

  • Programming Language: Rust
  • Build Tool: Cargo
  • Platform: Windows / Linux
  • Version Control: Git and GitHub
  • Interface: Command Line Interface (CLI)

Project Architecture

AutoFlow follows a simple and modular architecture. The program scans a given directory, identifies files based on their extensions, and maps them to predefined categories. Each category corresponds to a folder.

The application uses Rust’s standard library for file system operations such as reading directories, creating folders, and moving files.

How AutoFlow Works

  1. The user runs the AutoFlow executable from the command prompt
  2. The program scans the target directory
  3. File extensions are analyzed
  4. Folders are created if they do not already exist
  5. Files are moved into their respective folders

Command Line Usage

Example command:

autoflow.exe "C:\Users\sunil\OneDrive\Desktop\autoflow files"

This command organizes all files inside the specified directory.
